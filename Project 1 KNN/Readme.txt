To use the testlearner.py to test the learnenrs and generate the charts, you should take the following instructions.

1. Open Command Line.
2. 'cd FolderOfTestlearner.py'
3. 'python testlearner.py DataSetFileName'
4. Find the graphs(pdf files) under the same folder with testlearner.py

It will take several minites to generate the graphs, so please be patient.
